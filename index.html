System Description

NTN Handover Simulator is a Non-Terrestrial Network (NTN) simulation dashboard designed to visualize how emerging AI predictive models optimize User Equipment (UE) satellite handovers compared to standard, naive RSRP threshold-based switching. It continuously models orbital distance paths, free-space path loss, user equipment velocity constraints, and temporal fading natively in your browser.

Operating Principles

This simulator benchmarks a CatBoost-optimized handover logic against classic threshold-based policies. The ML model predicts transition viability based on RSRP, SINR, UE velocity, and elevation trajectories.

Inference Engine

Real-time inferences are resolved directly within the browser using ONNX Runtime Web. Live feature engineering, including RSRP moving averages and normalized temporal factors, feed into a precise ML decision boundary.

Glossary

RSRP: Reference signal received power.
SINR: Signal quality ratio under noise/interference.
Hysteresis: Minimum gain needed before switching serving satellite.
