Operating Principles

This simulator compares CatBoost-optimized handover decisions against classic RSS-based policies. The AI model accounts for RSRP, SINR, UE velocity, and elevation to reduce signal loss during orbital transitions.

Inference Engine

Real-time inference is handled by an ONNX model running directly in the browser environment. Feature normalization is performed by a dedicated preprocessing ONNX graph prior to classification.
