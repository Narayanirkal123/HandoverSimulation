System Description

NTN Handover Simulator is an advanced Non-Terrestrial Network (NTN) simulation dashboard designed to benchmark various AI predictive models against standard heuristic RSRP threshold-based handovers. It dynamically models independent orbital paths, free-space path loss, user equipment velocity constraints, and link-quality fading in real-time natively inside your browser.

Evaluation Architecture

This simulator operates in a multi-agent mode where CatBoost, XGBoost, and GradientBoost are each assigned their own sovereign User Equipment (UE). Each model is rigorously benchmarked against a distinct mathematical baseline mimicking its exact spatial trajectory for absolute fairness.

Inference Engine

Real-time multi-model inferences are resolved directly within the browser using ONNX Runtime Web. Live feature engineering—including RSRP moving averages, spatial distance vectors, and normalized temporal parameters—is processed synchronously and fed into each model's continuous decision loop.

Glossary

RSRP: Reference signal received power.
SINR: Signal quality ratio under noise/interference.
Hysteresis: Minimum gain needed before switching serving satellite.
